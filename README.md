# MedAgent HMS – Multi-Agent Healthcare Orchestration Platform

## Overview

MedAgent HMS is a full-stack healthcare orchestration platform designed to streamline clinical workflows using specialized AI agents.

The system enables intelligent coordination between healthcare operations, prescription risk analysis, and scheduling workflows.

## Features

* Multi-Agent orchestration architecture
* Pharmacy Agent for prescription risk analysis
* Scheduler Agent for automated clinic scheduling
* Real-time communication with WebSockets
* Clinical risk intelligence dashboard
* Responsive frontend built with Next.js and TypeScript

## Tech Stack

**Frontend**

* Next.js
* TypeScript
* Tailwind CSS

**Backend**

* Python
* WebSockets

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

http://localhost:3000
